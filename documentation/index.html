A.I) Installation

Infinity is HTML template so there's no really "installation" required just unzip the main file you've downloaded from Themeforest and after unziping you have your all template ready to be used.

I'm using one main CSS file in this theme.This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

			    <script src="js/prefixfree.min.js" type="text/javascript"></script>


			    
			    <link rel="stylesheet" href="css/theme-responsive.css" type="text/css" />


			    
			    <link rel="stylesheet/less" href="css/skins/skin.less" type="text/css" />
